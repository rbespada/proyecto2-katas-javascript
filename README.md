# Proyecto 2: Katas JavaScript

Repositorio correspondiente al Proyecto 2 del módulo de JavaScript.

## Estructura del proyecto

- ejercicios-basicos-avanzados/  
  Katas de JavaScript básico y avanzado.
- ejercicios-dom/  
  Ejercicios de manipulación del DOM (HTML + JS).
- ejercicios-es6-asincronia/  
  Ejercicios de ES6 y asincronía.

## Cómo ejecutar los ejercicios

### Ejercicios JavaScript (.js)
Desde la raíz del proyecto escribiremos en la terminal el siguiente comando, completando al final de la ruta con el ejercicio correspondiente que queramos ejecutar:

```bash
